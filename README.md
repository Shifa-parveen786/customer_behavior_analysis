# customer_behavior_analysis
This project is an interactive Power BI dashboard that analyzes customer purchasing behavior and provides meaningful business insights. It includes key performance indicators (KPIs), revenue analysis, sales trends, customer demographics, and subscription insights through dynamic visualizations.

Customer Behavior Dashboard

Overview
The Customer Behavior Dashboard is built using Microsoft Power BI to analyze customer shopping patterns and sales performance. It helps businesses understand customer behavior through interactive charts, KPIs, and filters.

Features
Total Number of Customers
Average Purchase Amount
Average Review Rating
Revenue by Category
Sales by Category
Revenue by Age Group
Sales by Age Group
Customer Subscription Analysis
Interactive Filters (Gender, Category, Subscription Status, Shipping Type)

Tools Used
Microsoft Power BI
Power Query
DAX
Excel Dataset

Dashboard Insights
Identifies top-performing product categories.
Compares sales and revenue across customer age groups.
Analyzes customer subscription trends.
Tracks average purchase amount and review ratings.

Project Goal
To transform raw customer shopping data into an interactive dashboard that supports data-driven business decisions.
